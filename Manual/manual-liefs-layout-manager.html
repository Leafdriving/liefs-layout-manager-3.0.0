Liefs-layout-manager   The Manual
Welcome To Liefs-Layout-Manager!

Liefs Layout Manager is a Javascript Framework (writen in Typescript) that manages the position of 'Div' Elements on a Page. The position of elements on the page is determined by using elements (Classes) of the framework, called by simple functions.

In essence, It ignores the inheritance of 'div' objects, and places them all on the first level of the dom, using a coordinate system. (x, y, width, height) and assigns these values in the 'style' tag of the element.

Most of the 'rules' of the framework are intuitive, and easy to learn by Novice, and Experienced Programmers Alike.

This Manual is written under this framework, and includes most of the features. For example, try re-sizing this window to see how the layout changes to fit all sizes. Try the Dragbar, and tree system, all managed by this framework.

There are TONS of examples included here to see each feature working in it's simpleist, to most difficult form.

Enjoy!

Installation

Lief Layout Manager is "installed" by simply adding one javascript file to your html file.(index.html probably)

load liefs-layout-managerV3.0.0.js or more likely, liefs-layout-managerV3.0.0.min.js

At time of posting, you can find the file under this repo:
https://github.com/Leafdriving/liefs-layout-manager-3.0.0/
or to be more percise, this file:
https://raw.githubusercontent.com/Leafdriving/liefs-layout-manager-3.0.0/master/js/liefs-layout-managerV3.0.0.js (Right-click -> save (as))

This is the basic boilerplate for your code:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Your Title</title>
    <script src="./{Your Path}/liefs-layout-managerV3.0.0.js"></script>
<!--      The above line loads the lief-layout-manager framework        -->
  </head>
  <body>
  <!-- Note - on invoking liefs-layout manager, all class="remove" are removed from the dom -->
  <div class="remove" style="font-size: larger;">Loading...</div>
  <!-- So until invoked, the user will see the above loading page -->
  <div class="remove" style="display:none">
  <!-- This is where you put all 'div' elements that you wish the handler to manage
        note, they too are removed, and re-introduced to the dom only when needed --> 
  </div>

  </body>
</html>


<!--  Your liefs-layout-manager code MUST come after the body to work! -->
<script src="{YourPath}/yourFrameworkCode.js"></script>
<!--  by adding a reference  to your framework script-->
<script>
H( {some code} ) // invoking a handler like this starts liefs-layout-manager framework
</script>
<!--  or just adding it to the END of your html -->
